Getting Started

Installation Time

This program is really light weight, in which the installation time for this will be much lower than others.

Install JasonScriptingUtilities to your Computer#

Follow the instructions below to install JasonScriptingUtilities into your computer. If you have any questions or issues, please join our Discord Server or Github Issues.

1. Installing From GitHub Releases#

Head over to our Github Releases and select the latest version. Install the .EXE file.

2. Locate the Directory#

Locate the directory of where you installed the client and open it up.

Congratulations!#

You have just installed JasonScriptingUtilities.

JasonScriptingUtilities has much more to offer!
Do not Move Anything

After installing this program, do not move any of the files inside of the program. This will cause issues.
